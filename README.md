# Catrace

一个帮你平衡工作与休息的小工具。

## 它是干嘛的

很多人一坐电脑前就是几个小时，等反应过来已经腰酸背痛了。Catrace 就是来解决这个问题的——它在后台默默观察你的活动状态，发现你连续工作太久，就会提醒你站起来休息一会儿。

## 怎么判断你在工作

Catrace 不会偷拍屏幕，也不会读取你在干嘛。它只是每隔几秒看看你的鼠标有没有动、键盘有没有敲。

**它的逻辑很简单：**

- 它从你今天第一次活动开始，按「工作窗口」向后切分时间段。
- 中间停下来回个消息、倒杯水、发个呆，只要没连续歇够 5 分钟，都不算打断当前工作窗口。
- 只有当你连续好几分钟（比如 5 分钟）完全没动静，它才认为你真正在休息，并切出一个新的休息 block。
- 如果你已经忙满了一个工作窗口（比如 45 分钟），中间一直没歇够 5 分钟，或者休息完后又工作满了一个窗口，它就会弹出一条通知，温柔地提醒你：该休息啦。

## 提醒方式

到时间后，它会通过系统通知提醒你。你可以设置自己的工作时长和休息判定时长，找最适合自己的节奏。

## 隐私

所有数据都存在你自己电脑里，不上传任何服务器。

## 界面概览

Catrace 提供简洁的 Dashboard，帮你回顾今天的工作与休息节奏：

- **今日统计**：活跃分钟、休息分钟、活跃占比、活跃时段数量
- **今日活动（概览）**：按工作窗口算法切分的时段列表，默认展示；点击条目可展开查看分钟级色块
- **今日活动（详细）**：24 小时 × 60 分钟的色块热力图，适合精细查看
- **设置**：调整工作窗口长度与休息打断阈值

界面采用淡紫 wellness 主题，侧边栏导航 + 主内容区布局。
