# 2026-07-23 单张 Toast 被自动滚动导致顶部裁切

## 现象

Toast 窗口只显示一张 Agent 卡片时，卡片顶部标题和圆角被窗口边界裁掉。

## 原因

`ReminderToast.vue` 每次新增通知后都会调用 `scrollStackToBottom()`。`toast-stack` 使用上下 `1rem` padding 给卡片阴影留空间；单张卡片也滚到底时，顶部 padding 会被卷出可视区，看起来像窗口高度不足或 UI 被截断。

插件 demo 每 10 秒产生 Toast，使窗口滚动状态更容易被观察到，但插件运行时没有修改 Agent 卡片样式或 Toast 尺寸计算。

## 修复

单张通知时强制 `scrollTop = 0`，保留顶部阴影空间；只有多张通知堆叠时才滚到底展示最新卡片。

- `src/views/ReminderToast.vue`
