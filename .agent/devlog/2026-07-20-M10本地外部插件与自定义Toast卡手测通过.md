## Key learnings（勿丢）

- 插件 UI **必须 Blob 加载**，不要依赖 file/asset 动态 import  
  详见 bug：`2026-07-20-插件ui动态import-file-asset失败改blob加载.md`
- 插件 ESM：**无 template 字符串、无 `import 'vue'`**
- 组件进 Pinia 前 **`markRaw`**
- Toast 窗与主窗各自 Pinia，都要 `loadExternalPlugins`
