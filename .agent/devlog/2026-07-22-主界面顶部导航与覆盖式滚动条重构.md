# 2026-07-22 主界面顶部导航与覆盖式滚动条重构

## Session goal

参考 `catrace.html` 将普通应用页面从左侧主导航重构为顶部导航，并让插件页保持左侧二级栏与右侧工作区。

## Completed

- 普通页面改为固定顶部导航，Reminder 独立窗口继续绕过普通应用外壳。
- 插件页保持左右分栏，左侧插件列表与右侧详情分别滚动。
- 新增覆盖式滚动容器：原生滚动条隐藏，悬停或拖动时显示浮层滑块，不占内容宽度。
- 覆盖式滚动样式只接入普通应用和插件区域，避免影响 Toast 的尺寸测量与透明窗口。

## Key file changes

| File | Change |
|------|--------|
| `src/App.vue` | 顶部导航、普通应用外壳与主工作区滚动容器 |
| `src/components/OverlayScrollbar.vue` | 覆盖式滚动条尺寸同步、悬停显示和拖动逻辑 |
| `src/views/Plugins.vue` | 固定左右分栏，并为插件列表和详情接入独立滚动容器 |

## Key conventions

- Reminder Popup、Fullscreen、Toast 不挂载 `.app-shell`，普通页面样式必须限定在该外壳内。
- 页面滚动优先使用 `OverlayScrollbar`，不要再给同一容器添加原生 `overflow-y: auto`。
- 插件页窄窗口仍保持左右分栏，仅收窄插件栏，避免列表和详情变成上下堆叠。
